How to Write Blog Posts#

This is a sample blog post to show you how to write content for your Hugo blog.

Writing Process#

1. Create the Post#

hugo new posts/your-post-title.md

2. Edit the Front Matter#

Update the metadata at the top:

  • title: Your post title
  • description: SEO description
  • tags: Array of tags
  • categories: Array of categories
  • draft: Set to false to publish

3. Write Your Content#

Use Markdown for formatting:

  • Bold text
  • Italic text
  • Links
  • Code snippets

4. Publish#

git add .
git commit -m "Add new post"
git push

Tips#

  • Use descriptive titles
  • Add relevant tags for organization
  • Write in Markdown for easy formatting
  • Preview locally with hugo server
